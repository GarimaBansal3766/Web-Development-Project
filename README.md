# Online Shopping Web Application 🛍️

An Online Shopping Web Application developed using ASP.NET MVC and SQL Server that provides a smooth and responsive shopping experience. The project implements CRUD operations for managing products and categories efficiently with proper database connectivity. It also includes secure Admin Login/Logout functionality for authentication and management purposes.

The frontend is designed using HTML, CSS, JavaScript, and Bootstrap to create an attractive, user-friendly, and responsive interface compatible with different devices and screen sizes. The application follows the MVC architecture to maintain organized, scalable, and maintainable code structure.

This project helped in gaining practical experience in full stack web development, database management, authentication systems, and responsive UI design using Microsoft technologies.

### Technologies Used

* ASP.NET MVC
* SQL Server
* HTML
* CSS
* JavaScript
* Bootstrap
